How to install ros package¶

1. ROSパッケージのリポジトリをクローンする¶

$git clone https://gitlab.com/trcp/trcp_patin.git

2. リポジトリに移動する¶

$cd trcp_patin

3. git submoduleで非公開パッケージの中身を追加する¶

$git submodule update --init --recursive

注釈

パスワードは聞いてください
